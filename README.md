# Single Page Resume

A simple and professional single-page resume website built using HTML5. This project showcases my education, skills, internship experience, and social profiles in a clean and structured format.

## Features

- Semantic HTML structure
- SEO Meta Tags
- Open Graph (OG) Tags for social media sharing
- Custom Favicon
- Skills Section
- Education Section
- Internship Experience
- GitHub and LinkedIn Links
- Single-page resume layout
